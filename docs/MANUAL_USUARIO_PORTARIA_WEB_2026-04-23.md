# Manual do Usuário
## Portaria Web

Versão: 1.0  
Data: 23/04/2026

Este manual foi preparado para orientar o uso do sistema `Portaria Web` de forma simples, prática e sem linguagem técnica.

---

## 1. Visão geral do sistema

O `Portaria Web` é a plataforma usada para acompanhar a rotina do condomínio, registrar cadastros, consultar informações e acompanhar a operação do dia a dia.

No sistema, cada perfil acessa apenas as áreas que fazem sentido para sua rotina.

Perfis mais comuns:

- `Master`: acompanha a visão geral dos clientes e módulos.
- `Admin`: gerencia o condomínio, pessoas, veículos, encomendas, câmeras e usuários.
- `Operação`: acompanha acessos, alertas, câmeras e ações operacionais.
- `Morador`: consulta suas informações, alertas, encomendas, veículos e perfil.

---

## 2. Como entrar no sistema

### Passo a passo

1. Abra o navegador.
2. Acesse o endereço do sistema.
3. Informe seu e-mail.
4. Informe sua senha.
5. Clique em `Entrar`.

### O que acontece depois do login

- Se o acesso estiver correto, o sistema abre o painel principal do seu perfil.
- Se houver erro de senha ou sessão, o sistema pedirá uma nova tentativa.

## 3. O que cada área faz

### Dashboard

O dashboard mostra um resumo visual do que está acontecendo no sistema.

Exemplos:

- quantidade de cadastros
- alertas do período
- encomendas
- acessos
- câmeras
- dispositivos

Os cards podem levar o usuário direto para a área correspondente.

### Moradores

Permite consultar, cadastrar, editar e visualizar os dados das pessoas vinculadas ao condomínio.

Também pode reunir:

- moradores
- visitantes
- prestadores
- locatários

### Unidades

Mostra as unidades do condomínio e ajuda no vínculo com moradores e cadastros.

### Usuários

Permite criar e administrar acessos ao sistema.

Exemplos:

- admin
- operação
- central
- morador

### Alertas

Mostra ocorrências importantes que precisam de acompanhamento.

Exemplos:

- alertas de monitoramento
- eventos operacionais
- ocorrências que exigem atenção

### Encomendas

Permite registrar, consultar e acompanhar encomendas até a retirada.

### Veículos

Permite cadastrar, consultar, editar e excluir veículos vinculados às unidades ou pessoas.

### Câmeras

Mostra a situação das câmeras cadastradas e a disponibilidade de imagem.

### Dispositivos

Mostra equipamentos de acesso e controle usados no condomínio.

### Operação

Área voltada ao uso do porteiro ou operação, com foco em:

- acompanhamento de acessos
- encomendas
- câmeras
- ocorrências
- busca por imagem

## 4.2. Cadastrar uma pessoa

1. Abra a área `Moradores`.
2. Clique em `Novo cadastro`.
3. Escolha o tipo de cadastro.
4. Preencha os dados pedidos.
5. Se houver foto, adicione a imagem.
6. Clique em `Salvar`.

Tipos mais comuns:

- morador
- visitante
- prestador
- locatário

## 4.3. Consultar uma pessoa

1. Abra a área `Moradores`.
2. Use a busca para localizar pelo nome ou outro dado disponível.
3. Clique no registro desejado.
4. Veja os detalhes na tela ou modal de consulta.

3. Escolha a unidade ou destinatário.
4. Informe os dados principais.
5. Adicione foto, se necessário.
6. Clique em `Salvar`.

## 4.8. Cadastrar ou editar veículo

1. Abra a área `Veículos`.
2. Clique em `Novo veículo` para cadastrar.
3. Preencha os dados do veículo.
4. Escolha o vínculo correto.
5. Clique em `Salvar`.

Para editar:

1. Localize o veículo na lista.
2. Clique em `Editar`.
3. Atualize os dados necessários.
4. Clique em `Salvar`.

Para excluir:

1. Localize o veículo.
2. Clique em `Excluir`.
3. Confirme a exclusão.

## 6. Dúvidas comuns

### O sistema atualiza sozinho?

Sim. O painel principal atualiza automaticamente em intervalos regulares.

### Posso clicar nos cards?

Sim. Os cards do dashboard podem abrir a área correspondente.

### O que fazer se uma tela não carregar?

1. Atualize a página.
2. Verifique sua conexão.
3. Entre novamente no sistema, se necessário.

### O que fazer se aparecer mensagem de permissão?

Esse aviso indica que o perfil logado não tem liberação para aquela ação. Nesse caso, a administração do sistema deve revisar o acesso.

### O modo tela cheia é obrigatório?

Não. Ele é apenas uma forma de acompanhar o painel com mais conforto.

## 7. Boas práticas de uso

- use sempre o perfil correto para cada rotina
- revise os dados antes de salvar
- evite duplicar cadastros
- confira unidade, pessoa e veículo antes de confirmar
- em caso de dúvida, consulte o histórico antes de agir
