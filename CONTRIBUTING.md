# Contribuicao

## Padrao de commits

Use commits pequenos e objetivos com os prefixos:

- `feat`
- `fix`
- `docs`
- `chore`

Exemplos:

```text
feat: adiciona painel de busca facial
fix: corrige cadastro de camera RTSP
docs: atualiza resumo da integracao v5.3
chore: organiza arquivos de autoria do repositorio
```

## Regras gerais

- Revise texto, acentuacao e experiencia do usuario antes de publicar
- Nao publique segredos, chaves, tokens ou credenciais
- Atualize `CHANGELOG.md` em releases relevantes
- Mantenha a autoria preservada nos documentos do projeto
